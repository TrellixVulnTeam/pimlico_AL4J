Embeddings
~~~~~~~~~~


.. py:module:: pimlico.modules.input.embeddings


Read vector embeddings (e.g. word embeddings) from various storage formats.

There are several formats in common usage and we provide readers for most of these here:
`FastText`, `word2vec` and `GloVe`.
